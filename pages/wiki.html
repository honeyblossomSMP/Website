How to Join

To join HoneyBlossomSMP:

  1. Join the Discord server
  2. Read the rules in #rules.
  3. Create a support ticket and request whitelisting.

Staff will process your ticket — check Discord for updates.

Player Commands (Sorted)

Organized player-focused commands. Replace or extend these with plugin-specific variants if needed.

1 - Teleportation

Player Teleporting

/tpa <player>
Request to teleport to a player
/tpahere <player>
Request for a player to teleport to you
/tpacancel
Cancel your teleport request
/tpaaccept
Accept your teleport request

Homes (Currently 1 Home)

/sethome <name>
Set a home
/delhome <name>
Delete a home
/home <name>
Teleport to your home

Warps

/pw
Open the Player Warps menu
/pw <name>
Teleport to that warp

2 - Claiming & Protection

Use a wooden shovel to claim: right-click one corner and then the opposite corner. View boundaries with a stick. You gain 175 claim blocks per hour of playtime.

/claim
Claim the land you are standing on
/unclaim
Remove your claim
/extendclaim <number>
Extend your claim in facing direction
/trust <player>
Allow a player to build inside your claim
/untrust <player>
Remove the player’s access
/containertrust <player>
Allow chest/container access
/accesstrust <player>
Allow door/button/use access
/permissiontrust <player>
Allow a trusted player to grant trusts below their level
/claimlist
List your claims & claim block usage
/trustlist
List players you have trusted and levels

3 - Jobs, Skills & Leveling

Economy

Overview of economy and job commands (EssentialsX / Jobs plugins may vary).

Jobs (Max 3)

Players can have up to 3 jobs simultaneously. Join jobs to earn money while you play.

/jobs browse
List all available jobs
/jobs join <job>
Join a job (alias: /job join <job> on some setups)
/jobs stats
View your job progress
/jobs leave <job>
Leave a job

Available Jobs (common on the server):

  • Beekeeper
  • Builder
  • Digger
  • Hunter
  • Explorer
  • Brewer
  • Crafter
  • Enchanter
  • Farmer
  • Fisherman
  • Miner
  • Weaponsmith
  • Woodcutter

Economy Commands

/bal
Show your money balance
/pay <player> <amount>
Send money to another player
/sell <item>
Sell an item at configured shop (plugin-dependent)
/shop
Open server shop GUI (if provided)

Shops, prices and job payouts are configured by admins — check Discord or in-game signs for public shop listings and payout info.

QuickShop

Player shop commands (server uses QuickShop-style commands):

/qs create <price>
Create a shop (usually place chest/sign then run this)
/qs buy <shopId> <amount>
Buy items from a QuickShop
/qs sell <shopId> <amount>
Sell items to a QuickShop
/qs remove
Remove your shop
/qs info <shopId>
Show information about a shop

Exact command syntax can vary by configuration — run /help qs or check Discord for full docs.

Fadah Auction House

Players only need the following command to use the auction house:

/ah
Open the auction house GUI (player-facing command)

NOrder

NOrder allows players to place item orders/requests on the market.

/order <item> <amount>
Place an order/request for an item (server may require shop funds or specific syntax)

How to Join

Steps to join the server:

  1. Add the server IP: honeyblossomsmp.org in your Minecraft multiplayer list.
  2. Make sure your client version matches the server (1.21.x).
  3. If the server is whitelisted, join the Discord and request access from staff.
  4. Read the rules in Discord before building.

Server Voting

Support the server by voting on server lists. Common flow:

  • Visit the server vote page(s) linked in Discord or on the website.
  • Complete the captcha / voting action.
  • Return in-game and run /vote

Rewards often include in-game currency, keys, or temporary perks — check the voting page for details.
